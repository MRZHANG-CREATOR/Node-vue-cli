# Node-vue-cli🚀🚀🚀🚀🚀

### 切换到当前文件夹

```

cd Node-vue-cli

```

### 全局链接

```

npm  link

```

### 拉取vue基本框架

```

zpp  init  项目名称

```
### 自动安装依赖

```

npm install(自动)

```
### 自启动项目

```

npm  run serve(自动)

```
